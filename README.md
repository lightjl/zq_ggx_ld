华泰的这个策略是这样的：
    1） 总市值全市场从大到小前80%
    2）市盈率全市场从小到大前40%（剔除市盈率为负的股票）
    3）市收率小于2.5
    4）同时满足上述3条的股票，按照股息率从大到小排序，选出股息率最高的50只股票

市收率=当前价格/最近12个月的每股销售收入
